CSS selectors

Types of selectors Example Description
Type selectors h1 Type selectors target an HTML element
Class selectors .example-container Class selectors target an element or multiple element's that have a specific value for their class attributes
ID-selectors #first-heading ID selectors target an element that has a specific value for its id attribute
Attribute selectors img[alt] This group of selectors gives you different ways to select elements based on the presence of a certain attribute within an HTML element
Attribute + value selectors img[alt="grizzly-bear"] You can also make a selection based on the value which has been set within the specific attribute, allowing for a more precise selection
Pseudo-class selectors a:hover These group of selectors style certain states of an element. The :hover pseudo-class selects an element only when it is being hovered over by the mouse pointer
Universal selector * This target will select all HTML elements on the page
Grouping selectors h1, h2, p Instead of making multiple rulesets for different elements with the same containing the same declarations, you can merge all the selectors into a group selector. This is a more effective way for applying the same style on the desired elements
Chain selectors p.intro If you want to only target <p> elements with certain classes for example, you can do so with chain selectors. These selectors will narrow down your selection as precise as desired and only apply the style to the elements that match all criteria within the selector

As you become more proficient in CSS, you'll discover a wide array of advanced selectors that allow you to target specific elements with greater precision and flexibility. These advanced selectors go beyond basic element, class, and ID selectors, offering powerful tools to style your web pages more efficiently.

Advanced selectors enable you to target elements based on their position, relationship with other elements, attributes, and even their content. By mastering these selectors, you can create more complex and dynamic styles while writing cleaner and more maintainable CSS code.

CSS specificity

Types of selectors Example Specificity Value
ID-selectors #first-heading 100
Class- attribute- and pseudo-class selectors .box, img[alt], .box:hover 10
Elements and pseudo-elements p, button:hover 1
Universal selector and inherited styling *, p style from body ruleset -

CSS specificity is the mechanism by which browsers decide which CSS rules should take precedence when multiple rules apply to the same element. Specificity is calculated based on the types of selectors used in a CSS rule.
